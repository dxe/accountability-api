API used by the Stakeholders Accountability web app.

# Setting up locally 
1. Clone this repo.
2. ```npm install```
3. Set up .env file:

    ```
    PORT=XXXX
    DATABASE_URL=XXXX
    SECRET=XXXX
    GOOGLE_OAUTH_CLIENT=XXXX
    TWILIO_SID=XXXX
    TWILIO_TOKEN=XXXX
    TWILIO_SERVICE=XXXX
    ```
    
4. ```npm start```

# Deploying changes
Changes pushed to master are automatically deployed.
